\CVItem{Differenciator}{
Derivative calculator with step-by-step solution. Creates AST for

Link: \url{https://github.com/victorbaldin56/differenciator}
}

\CVItem{Miniprintf}{
Simplified implementation of \texttt{printf} function in pure NASM assembly for x86-64
Linux.\\
Link: \url{https://github.com/victorbaldin56/miniprintf}
}

\CVItem{Mandelbrot}{
Mandelbrot fractal
renderer built with use of SFML. The main goal of this project was the
research of SIMD instructions to speed up calculations up to several times. On my machine
I have reached 7.4 times acceleration with use of AVX2 instructions.\\
Link: \url{https://github.com/victorbaldin56/mandelbrot_fractal}
}

\CVItem{Hash Map}{
Accelerated implementation of hash map.
Educational project to master skills of agressive platform-dependent optimizations with intrinsics and assembly.
As the result, search by key has been speeded up by 2.8 times.\\
Link: \url{https://github.com/victorbaldin56/hashmap}
}

\CVItem{Binary Translator}{
Translates my CPU emulator's bytecode to real x86 opcodes.
Supports generating NASM assembly or JIT.\\
Link: \url{https://github.com/victorbaldin56/binary_translator}
